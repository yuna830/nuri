# 기초생활보장제도

## 1. 한 줄 요약

기초생활보장제도는 생활이 어려운 저소득층에게 생계급여, 의료급여, 주거급여, 교육급여 등을 지원하여 최저생활을 보장하고 자립을 돕는 제도이다.

## 2. 지원 대상

기초생활보장제도는 다음 기준을 충족하는 저소득층을 대상으로 한다.

- 소득인정액이 급여별 선정기준 이하인 사람
- 생계급여 선정기준 이하인 가구
- 의료급여 선정기준 이하인 가구
- 주거급여 선정기준 이하인 가구
- 교육급여 선정기준 이하인 가구
- 의료급여의 경우 부양의무자 기준을 충족해야 할 수 있음
- 생활 유지가 어렵고 공적 지원이 필요한 가구

기초생활보장제도는 하나의 기준으로 모든 급여를 결정하는 방식이 아니라, 생계급여, 의료급여, 주거급여, 교육급여별로 선정기준이 다르다.

## 3. 주요 목적

기초생활보장제도의 주요 목적은 다음과 같다.

- 최저생활 보장
- 생계 유지 지원
- 의료비 부담 완화
- 주거 안정 지원
- 교육 기회 보장
- 저소득층의 생활 안정
- 빈곤으로 인한 위기 예방
- 자립 기반 마련

## 4. 맞춤형 급여의 의미

맞춤형 기초생활보장제도는 기존 통합급여를 급여별로 나누어 지원하는 방식이다.

| 급여 종류 | 의미 |
|---|---|
| 생계급여 | 기본적인 생활비 지원 |
| 의료급여 | 병원 진료와 치료비 부담 완화 |
| 주거급여 | 임차료, 주택수선 등 주거비 지원 |
| 교육급여 | 학생의 교육활동비 등 교육비 지원 |

가구의 소득인정액이 어느 기준 이하인지에 따라 받을 수 있는 급여가 달라질 수 있다.

예를 들어 생계급여 기준은 넘지만 주거급여 기준 이하라면 주거급여 대상이 될 수 있다.

## 5. 소득인정액 의미

소득인정액은 실제 소득만 의미하지 않는다.

소득인정액은 다음 항목을 종합하여 계산한다.

- 근로소득
- 사업소득
- 재산소득
- 공적이전소득
- 금융재산
- 부동산 등 일반재산
- 자동차
- 부채
- 재산을 소득으로 환산한 금액

즉, 월 소득이 적어도 재산이 많으면 소득인정액이 높게 계산될 수 있다.

## 8. 생계급여

생계급여는 생활을 유지하기 어려운 가구에 기본적인 생활비를 지원하는 급여이다.

주요 내용은 다음과 같다.

- 생계 유지에 필요한 현금 지원
- 소득인정액이 생계급여 선정기준 이하인 가구 대상
- 급여액은 선정기준액과 가구의 소득인정액 차이를 기준으로 산정될 수 있음
- 매월 지급되는 방식으로 운영될 수 있음

생계급여는 최저생활 보장을 위한 가장 기본적인 급여이다.

## 9. 의료급여

의료급여는 생활이 어려운 저소득층의 의료비 부담을 줄이기 위한 급여이다.

주요 내용은 다음과 같다.

- 진찰
- 검사
- 치료
- 입원
- 수술
- 약제
- 의료서비스 이용 지원

의료급여는 다른 급여와 달리 부양의무자 기준이 적용될 수 있으므로 별도 확인이 필요하다.

## 10. 주거급여

주거급여는 저소득층의 주거 안정을 돕기 위한 급여이다.

주요 내용은 다음과 같다.

- 임차가구 임차료 지원
- 자가가구 주택수선 지원
- 주거비 부담 완화
- 주거환경 개선 지원

주거급여는 소득인정액과 가구원 수, 거주 형태, 임차료 수준 등에 따라 지원 내용이 달라질 수 있다.

## 11. 교육급여

교육급여는 저소득층 학생의 교육비 부담을 줄이기 위한 급여이다.

주요 내용은 다음과 같다.

- 교육활동지원비
- 입학금
- 수업료
- 교과서비
- 교육 관련 비용 지원

교육급여는 학생이 있는 가구에서 중요하게 확인해야 하는 급여이다.

## 12. 신청 방법

기초생활보장제도는 읍·면·동 주민센터 또는 행정복지센터에 신청할 수 있다.

일반적인 신청 흐름은 다음과 같다.

1. 거주지 또는 가까운 읍·면·동 주민센터에 문의한다.
2. 신청서를 작성한다.
3. 소득, 재산, 가구원 정보를 제출한다.
4. 시·군·구 통합조사팀에서 조사를 진행한다.
5. 보장 여부를 결정한다.
6. 급여 대상자로 결정되면 급여가 지급된다.
7. 이후 소득, 재산, 가구 상황 변동 시 신고한다.

주민등록상 주소지와 무관하게 전국 읍·면·동에서 신청할 수 있는 것으로 안내되어 있다.

## 13. 문의처

기초생활보장제도는 급여 종류에 따라 문의처가 다를 수 있다.

정확한 상담은 거주지 행정복지센터 또는 보건복지상담센터를 통해 확인하는 것이 좋다.

## 14. 서비스에서 활용할 수 있는 판단 기준

우리 서비스에서는 기초생활보장제도 대상 가능성을 다음 기준으로 안내할 수 있다.

### 대상 가능성 높음

- 소득이 매우 낮음
- 재산이 적음
- 생계 유지가 어려움
- 의료비 부담이 큼
- 월세 또는 주거비 부담이 큼
- 학생 자녀의 교육비 부담이 큼
- 기초생활수급 이력이 있음
- 실직, 질병, 장애, 고령 등으로 생활 유지가 어려움

### 확인 필요

- 소득인정액 계산 필요
- 가구원 수 확인 필요
- 재산 수준 확인 필요
- 자동차 보유 여부 확인 필요
- 부양의무자 기준 확인 필요
- 급여별 선정기준 확인 필요
- 기존에 다른 복지급여를 받고 있는지 확인 필요
- 소득 또는 재산 변동 여부 확인 필요

### 대상 가능성 낮음

- 소득인정액이 모든 급여 선정기준을 초과함
- 재산이 많음
- 고가 자동차 등 환산 재산이 큼
- 의료급여 부양의무자 기준을 충족하지 못함
- 생계 어려움이 일시적이고 긴급복지지원이 더 적합한 경우

## 15. 서비스 입력 항목 예시

기초생활보장제도 추천 또는 상담 기능을 만들 때 다음 입력 항목을 활용할 수 있다.

| 항목 | 설명 |
|---|---|
| householdSize | 가구원 수 |
| monthlyIncome | 월 소득 |
| recognizedIncome | 소득인정액 |
| propertyAmount | 재산 수준 |
| financialAsset | 금융재산 |
| realEstateAsset | 부동산 재산 |
| vehicleOwned | 자동차 보유 여부 |
| debtAmount | 부채 |
| employmentStatus | 취업 상태 |
| healthStatus | 건강 상태 |
| disabilityStatus | 장애 여부 |
| elderlyHousehold | 고령 가구 여부 |
| medicalCostBurden | 의료비 부담 |
| rentBurden | 주거비 부담 |
| studentInHousehold | 학생 자녀 여부 |
| currentBenefits | 현재 받고 있는 복지급여 |
| familySupportAvailable | 가족 지원 가능 여부 |
| obligationProviderCheck | 부양의무자 기준 확인 필요 여부 |

이 항목들은 실제 수급 여부를 확정하는 것이 아니라, 행정복지센터 상담 전 사전 확인용으로 활용한다.

## 16. 간단 판단 로직

기초생활보장제도 대상 가능성은 다음 순서로 판단할 수 있다.

1. 가구원 수를 확인한다.
2. 월 소득과 재산을 확인한다.
3. 소득인정액을 계산하거나 추정한다.
4. 소득인정액을 급여별 선정기준과 비교한다.
5. 생계급여 기준 이하인지 확인한다.
6. 의료급여 기준 이하인지 확인한다.
7. 주거급여 기준 이하인지 확인한다.
8. 교육급여 기준 이하인지 확인한다.
9. 의료급여는 부양의무자 기준을 추가로 확인한다.
10. 신청 가능성이 있으면 읍·면·동 행정복지센터 신청을 안내한다.
11. 주거비 문제가 크면 주거급여 상담을 함께 안내한다.
12. 의료비 문제가 크면 의료급여 또는 긴급복지 의료지원도 함께 확인한다.

최종 대상 여부는 시·군·구 조사와 보장 결정을 기준으로 해야 한다.

## 17. 사용자 질문 예시

사용자가 다음과 같이 질문할 수 있다.

- 기초생활보장제도가 뭐야?

- 생계급여 받을 수 있어?
- 의료급여랑 생계급여 차이가 뭐야?
- 주거급여 신청할 수 있어?
- 교육급여는 누가 받을 수 있어?
- 소득인정액이 뭐야?
- 기준 중위소득이 뭐야?
- 월 소득이 적으면 기초생활수급자 가능해?
- 재산이 있으면 기초생활수급자 안 돼?
- 부양의무자 기준이 뭐야?
- 어디에서 신청해?

- 1인 가구 기준은 얼마야?
