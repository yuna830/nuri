# 주거급여

## 1. 한 줄 요약

주거급여는 소득인정액이 주거급여 선정기준 이하인 저소득층 가구에게 임차료 또는 주택수선비를 지원하여 주거비 부담을 줄이고 주거 안정을 돕는 제도이다.

## 2. 지원 대상

주거급여는 다음 조건을 충족하는 가구를 대상으로 한다.

- 소득인정액이 주거급여 선정기준 이하인 가구
- 임차주택에 거주하며 임차료 부담이 있는 가구
- 자가주택에 거주하지만 주택이 노후되어 수선이 필요한 가구
- 주거비 부담으로 생활 유지가 어려운 저소득층
- 기초생활보장제도 급여 중 주거급여 기준에 해당하는 가구

주거급여는 부양의무자의 소득이나 재산과 관계없이 신청 가구의 소득과 재산을 중심으로 판단한다.

## 3. 주요 목적

주거급여의 주요 목적은 다음과 같다.

- 저소득층의 주거비 부담 완화
- 안정적인 주거생활 지원
- 임차가구의 월세 또는 전월세 부담 완화
- 자가가구의 노후 주택 수선 지원
- 주거환경 개선
- 주거 불안정 예방
- 기초생활보장제도의 맞춤형 급여 지원
- 주거 취약계층의 생활 안정 지원

## 4. 주거급여의 종류

주거급여는 거주 형태에 따라 크게 두 가지로 나눌 수 있다.

| 구분 | 대상 | 지원 내용 |
|---|---|---|
| 임차급여 | 전세, 월세 등 타인의 주택에 거주하는 가구 | 실제 임차료를 기준으로 임차료 지원 |
| 수선유지급여 | 본인 소유 주택에 거주하는 가구 | 주택 노후도에 따라 수선비 지원 |

임차가구와 자가가구는 지원 방식이 다르므로 거주 형태를 먼저 확인해야 한다.

## 5. 2025년 주거급여 선정기준

2025년 주거급여 선정기준은 기준 중위소득의 48% 수준으로 안내된다.

| 가구원 수 | 2025년 주거급여 선정기준 |
|---:|---:|
| 1인 가구 | 1,148,166원 |
| 2인 가구 | 1,887,676원 |
| 3인 가구 | 2,412,169원 |
| 4인 가구 | 2,926,931원 |
| 5인 가구 | 3,411,932원 |
| 6인 가구 | 3,871,106원 |
| 7인 가구 | 4,314,445원 |

선정기준은 매년 바뀔 수 있으므로 실제 신청 시에는 해당 연도 기준을 다시 확인해야 한다.

## 6. 소득인정액 의미

소득인정액은 단순 월급만 의미하지 않는다.

소득인정액은 다음 항목을 종합하여 계산한다.

- 근로소득
- 사업소득
- 재산소득
- 공적이전소득
- 금융재산
- 부동산 등 일반재산
- 자동차
- 부채
- 재산을 소득으로 환산한 금액

즉, 실제 월소득이 낮아도 재산이 많으면 소득인정액이 높게 계산될 수 있다.

## 7. 임차급여 지원 내용

임차급여는 전세, 월세 등 타인의 주택에 거주하는 가구에게 임차료를 지원하는 방식이다.

주요 내용은 다음과 같다.

- 지역별 기준임대료를 상한으로 지원
- 가구원 수에 따라 기준임대료가 달라짐
- 실제 임차료를 기준으로 지원
- 소득인정액이 생계급여 선정기준을 초과하면 자기부담분이 차감될 수 있음
- 임대차계약서와 실제 거주 여부 확인 필요

임차급여는 실제 월세 전액을 무조건 지원하는 것이 아니라, 기준임대료와 실제 임차료, 소득인정액을 함께 반영하여 산정된다.

## 8. 실제 임차료 의미

실제 임차료는 월세만 의미하지 않는다.

전세보증금이나 월세보증금이 있는 경우 보증금을 월 임차료로 환산하여 계산할 수 있다.

일반적으로 확인하는 항목은 다음과 같다.

- 월세
- 전세보증금
- 월세보증금
- 임대차계약서상 임차료
- 실제 거주 여부
- 임대차계약 관계

임대차계약서가 없거나 실제 임차료가 0원인 경우에는 지급이 제한될 수 있다.

## 9. 수선유지급여 지원 내용

수선유지급여는 자가주택에 거주하는 주거급여 수급자에게 주택 수선을 지원하는 방식이다.

주요 내용은 다음과 같다.

- 주택 노후도에 따라 수선 범위 결정
- 경보수, 중보수, 대보수 등으로 구분 가능
- 도배, 장판, 창호, 단열, 난방, 지붕, 욕실 등 주택 상태에 따라 수선 가능
- 현금 지급이 아니라 주택 수선 방식으로 지원될 수 있음
- 실제 주택 상태 조사가 필요함

자가가구의 경우 월세 지원이 아니라 주택 개보수 지원 중심으로 운영된다.

## 10. 주택조사

주거급여는 신청 후 주택조사가 진행될 수 있다.

주택조사에서는 다음 사항을 확인한다.

### 임차가구 조사 항목

- 임대차계약 관계
- 실제 거주 여부
- 주택 현황
- 실제 임차료
- 보증금과 월세
- 주거 상태

### 자가가구 조사 항목

- 실제 거주 여부
- 주택 소유 여부
- 주택 노후도
- 수선 필요 정도
- 주택 구조와 안전 상태
- 주거환경 상태

주택조사는 주거급여 대상 여부와 지원 내용을 판단하는 데 중요한 절차이다.

## 11. 신청 방법

주거급여는 읍·면·동 주민센터 또는 행정복지센터에서 신청할 수 있다.

일반적인 신청 흐름은 다음과 같다.

1. 거주지 또는 가까운 읍·면·동 행정복지센터에 문의한다.
2. 주거급여 신청서를 작성한다.
3. 가구원, 소득, 재산, 주거 형태 관련 서류를 제출한다.
4. 소득과 재산 조사가 진행된다.
5. LH 등 담당기관의 주택조사가 진행될 수 있다.
6. 보장 여부와 급여액이 결정된다.
7. 임차급여 또는 수선유지급여가 지원된다.
8. 소득, 재산, 주거 상황이 바뀌면 변동사항을 신고한다.

온라인으로는 복지로를 통해 신청할 수 있는 경우도 있다.

## 12. 문의처

주거급여는 다음 기관에 문의할 수 있다.

| 구분 | 문의처 |

| 신청 상담 | 읍·면·동 행정복지센터 |
| 복지 상담 | 보건복지상담센터 129 |

지역별 세부 기준과 신청 가능 여부는 행정복지센터에서 확인하는 것이 가장 정확하다.

## 13. 신청 전 확인할 사항

신청 전에는 다음 항목을 확인하는 것이 좋다.

- 가구원 수
- 월 소득
- 재산 수준
- 소득인정액
- 임차가구 여부
- 자가가구 여부
- 월세 또는 전세보증금
- 임대차계약서 보유 여부
- 실제 거주 여부
- 주택 소유 여부
- 주택 노후도
- 기존 기초생활보장 수급 여부
- 현재 받고 있는 복지급여
- 주거비 체납 여부
- 이사 예정 여부

## 14. 서비스에서 활용할 수 있는 판단 기준

우리 서비스에서는 주거급여 대상 가능성을 다음 기준으로 안내할 수 있다.

### 대상 가능성 높음

- 소득인정액이 주거급여 선정기준 이하임
- 월세 부담이 큼
- 전세보증금 또는 월세보증금 부담이 큼
- 임대차계약서를 보유하고 있음
- 실제 임차주택에 거주함
- 자가주택이 노후되어 수선이 필요함
- 고령자, 장애인, 저소득층 등 주거 취약성이 큼
- 주거비 때문에 생계가 어려움

### 확인 필요

- 소득인정액 계산 필요
- 임대차계약서 확인 필요
- 실제 거주 여부 확인 필요
- 주택 소유 여부 확인 필요
- 주택 노후도 조사 필요
- 가구원 수 확인 필요
- 기준임대료 확인 필요
- 생계급여 선정기준 초과 여부 확인 필요
- 기존 기초생활보장 수급 여부 확인 필요

### 대상 가능성 낮음

- 소득인정액이 주거급여 선정기준을 초과함
- 임차료 부담이 없음
- 실제 거주하지 않음
- 임대차계약서가 없고 실제 임차료 확인이 어려움
- 자가주택 수선 필요성이 낮음
- 주거급여보다 긴급복지 주거지원이 더 적합한 단기 위기 상황

## 15. 서비스 입력 항목 예시

주거급여 추천 또는 상담 기능을 만들 때 다음 입력 항목을 활용할 수 있다.

| 항목 | 설명 |
|---|---|
| householdSize | 가구원 수 |
| monthlyIncome | 월 소득 |
| recognizedIncome | 소득인정액 |
| propertyAmount | 재산 수준 |
| housingType | 임차, 자가, 무상거주 등 거주 형태 |
| rentAmount | 월 임차료 |
| depositAmount | 보증금 |
| leaseContract | 임대차계약서 보유 여부 |
| actualResidence | 실제 거주 여부 |
| region | 거주 지역 |
| standardRentArea | 기준임대료 급지 |
| homeOwnership | 주택 소유 여부 |
| houseCondition | 주택 노후도 |
| repairNeed | 수선 필요 여부 |
| currentBenefits | 현재 받고 있는 복지급여 |
| housingCostBurden | 주거비 부담 정도 |
| rentArrears | 임차료 체납 여부 |
| movingPlan | 이사 예정 여부 |

이 항목들은 실제 수급 여부를 확정하지 않고, 행정복지센터 상담 전 사전 확인용으로 활용한다.

## 16. 간단 판단 로직

주거급여 대상 가능성은 다음 순서로 판단할 수 있다.

1. 가구원 수를 확인한다.
2. 소득과 재산을 확인한다.
3. 소득인정액을 계산하거나 추정한다.
4. 소득인정액을 주거급여 선정기준과 비교한다.
5. 거주 형태가 임차인지 자가인지 확인한다.
6. 임차가구라면 임대차계약서와 실제 임차료를 확인한다.
7. 자가가구라면 주택 소유 여부와 노후도를 확인한다.
8. 실제 거주 여부를 확인한다.
9. 주거비 부담 정도를 확인한다.
10. 신청 가능성이 있으면 행정복지센터 또는 복지로 신청을 안내한다.
11. 주택조사가 필요할 수 있음을 안내한다.
12. 단기 주거 위기라면 긴급복지 주거지원도 함께 확인한다.

최종 대상 여부와 지원 금액은 소득·재산 조사와 주택조사 결과를 기준으로 결정된다.

## 17. 사용자 질문 예시

사용자가 다음과 같이 질문할 수 있다.

- 주거급여가 뭐야?
- 월세 지원받을 수 있어?
- 주거급여 신청할 수 있어?
- 1인 가구 주거급여 기준이 얼마야?
- 임대차계약서가 있어야 해?
- 전세 살아도 주거급여 받을 수 있어?
- 자가주택도 주거급여 받을 수 있어?
- 집수리 지원받을 수 있어?
- 주거급여는 어디에서 신청해?
- 주거급여 콜센터 번호 알려줘
- LH 주택조사가 뭐야?
- 소득이 어느 정도면 주거급여 받을 수 있어?
- 기초생활수급자가 아니어도 주거급여 받을 수 있어?
- 월세가 밀렸는데 주거급여로 해결돼?

## 18. 관련 복지정책 및 RAG 검색 키워드

### 관련 복지정책

주거급여와 함께 확인하면 좋은 복지정책은 다음과 같다.

- 기초생활보장제도
- 생계급여
- 의료급여
