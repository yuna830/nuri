# 문화누리카드

## 1. 한 줄 요약

문화누리카드는 기초생활수급자와 차상위계층의 문화예술, 국내여행, 체육활동 이용을 지원하기 위해 1인당 일정 금액을 카드 형태로 지원하는 통합문화이용권이다.

## 2. 지원 대상

문화누리카드는 다음 사람을 대상으로 한다.

- 6세 이상 기초생활수급자
- 6세 이상 차상위계층
- 생계급여 수급자
- 의료급여 수급자
- 주거급여 수급자
- 교육급여 수급자
- 조건부 수급자
- 보장시설 수급자
- 차상위자활근로자
- 장애수당 수급자
- 장애아동수당 수급자
- 장애인연금 부가급여 수급자
- 본인부담경감대상자
- 저소득 한부모가족
- 차상위계층확인서 발급자
- 교육급여 수급자 외 나머지 가구원

2026년 기준으로는 2020년 12월 31일 이전 출생자가 대상이다.

## 3. 주요 목적

문화누리카드의 주요 목적은 다음과 같다.

- 저소득층의 문화생활 기회 확대
- 문화예술 접근성 향상
- 국내여행 기회 제공
- 체육활동 참여 지원
- 문화 격차 완화
- 삶의 질 향상
- 사회적 고립 완화
- 여가활동 지원
- 지역 문화시설 이용 촉진

## 4. 지원 금액

2026년 문화누리카드 지원 금액은 다음과 같다.

| 구분 | 지원 내용 |
|---|---:|
| 기본 지원금 | 1인당 연 150,000원 |
| 생애주기별 추가 지원금 | 1인당 10,000원 추가 |
| 추가 지원 대상 | 청소년기, 준고령기 |
| 청소년기 기준 | 2008년~2013년 출생자 |
| 준고령기 기준 | 1962년~1966년 출생자 |

단, 기본지원금 예산이 소진되면 카드 발급 또는 재충전이 불가할 수 있다.

생애주기별 추가지원금도 예산이 소진되면 기본지원금만 지급될 수 있다.

## 5. 발급 기간

2026년 문화누리카드 발급 기간은 다음과 같다.

| 구분 | 기간 |
|---|---|
| 발급 기간 | 2026년 2월 2일 ~ 2026년 11월 30일 |
| 주민센터 발급 | 2026년 11월 30일까지 |
| 온라인 발급 | 2026년 11월 30일까지 |
| 모바일앱 발급 | 2026년 11월 30일까지 |

주민등록주소지의 해당 연도 예산이 소진되면 조기 마감될 수 있다.

## 6. 사용 기간

2026년 문화누리카드 사용 기간은 다음과 같다.

| 구분 | 기간 |
|---|---|
| 카드 사용 기간 | 2026년 2월 2일 ~ 2026년 12월 31일 |

사용 기간이 끝나면 남은 잔액은 다음 해로 이월되지 않을 수 있으므로 기간 내 사용해야 한다.

## 7. 사용 가능 분야

문화누리카드는 문화예술, 국내여행, 체육 분야에서 사용할 수 있다.

| 분야 | 사용 예시 |
|---|---|
| 문화예술 | 공연, 영화, 전시, 도서, 음악, 사진관 등 |
| 국내여행 | 철도, 시외버스, 고속버스, 숙박, 관광지, 여행상품 등 |
| 체육활동 | 스포츠 관람, 체육용품, 체육시설 이용 등 |

사용 가능 여부는 반드시 문화누리카드 가맹점인지 확인해야 한다.

## 9. 사용 제한

문화누리카드는 모든 곳에서 사용할 수 있는 일반 체크카드가 아니다.

다음과 같은 곳에서는 사용이 제한될 수 있다.

- 일반 음식점
- 카페
- 마트
- 편의점
- 생활용품점
- 병원
- 약국
- 현금 인출
- 상품권 구매
- 문화·여행·체육과 무관한 업종
- 문화누리카드 가맹점이 아닌 곳

문화누리카드는 지정된 문화예술, 여행, 체육 가맹점에서만 사용하는 것이 원칙이다.

## 10. 신청 방법

문화누리카드는 다음 방법으로 신청할 수 있다.

| 신청 방법 | 내용 |
|---|---|
| 주민센터 방문 | 가까운 주민센터에서 신청서 작성 후 현장 수령 |
| 문화누리카드 누리집 | 온라인 신청 후 우편수령 또는 농협 영업점 방문수령 |
| 모바일 앱 | 문화누리카드 앱에서 신청 |
| 전화 ARS | 기존 카드 재충전 가능 대상자에 한해 전화 재충전 가능 |

온라인 신청은 본인인증이 필요하다.

농협 영업점 방문수령은 신규 발급의 경우 가능하며, 방문 전 공카드 수량 확인이 필요하다.

## 11. 주민센터 방문 발급 절차

주민센터 방문 발급 절차는 다음과 같다.

1. 주민센터를 방문한다.
2. 카드신청서를 작성하고 제출한다.
3. 주민센터에서 발급자격을 검증한다.
4. 상세정보를 등록한다.
5. 문화누리카드를 현장에서 직접 수령한다.

디지털 사용이 어렵거나 본인인증이 어려운 경우 주민센터 방문 신청이 가장 편할 수 있다.

## 12. 온라인 및 앱 발급 절차

문화누리카드 누리집 또는 모바일 앱 발급 절차는 다음과 같다.

1. 문화누리카드 누리집 또는 모바일 앱에 접속한다.
2. 발급자격을 검증한다.
3. 본인인증을 진행한다.
4. 상세정보를 등록한다.
5. 우편수령 또는 농협 영업점 방문수령을 선택한다.
6. 우편수령은 신청 완료 후 약 3~5일이 소요될 수 있다.
7. 농협 영업점 방문수령은 신청 완료 약 2시간 이후 가능할 수 있다.

농협 영업점 방문수령 시 신분증을 지참해야 한다.

## 13. 전화 ARS 재충전

기존 문화누리카드를 가지고 있는 사람 중 조건을 충족하는 경우 전화 ARS로 재충전할 수 있다.

전화 재충전 흐름은 다음과 같다.

1. 문화누리카드 고객지원센터 1544-3412에 전화한다.
2. 자동음성 안내에 따라 정보를 입력한다.
3. 자격 검증을 진행한다.
4. 본인 인증을 진행한다.
5. 재충전 완료 문자를 받는다.

전화 ARS 재충전은 14세 이상, 유효기간이 일정 기간 이후인 카드, 본인 명의 휴대전화 보유 등 조건이 필요할 수 있다.

복지시설 거주자나 일부 알뜰폰 사용자는 전화 재충전이 제한될 수 있다.

## 14. 자동재충전

자동재충전은 전년도 문화누리카드 이용자 중 일정 조건을 충족한 사람에게 별도 신청 없이 해당 연도 지원금을 충전해주는 방식이다.

2026년 자동재충전 대상은 다음 기준이 적용될 수 있다.

- 2025년에 문화누리카드를 발급받은 사람
- 2025년에 3만 원 이상 사용 이력이 있는 사람
- 2026년에도 수급자격을 유지하는 사람
- 자동재충전 제외 사유가 없는 사람

자동재충전이 완료되면 문자 안내가 발송될 수 있다.

자동재충전이 되지 않은 경우 주민센터, 누리집, 모바일 앱, 전화 ARS를 통해 신규발급 또는 재충전을 신청할 수 있다.

## 15. 잔액 확인 및 관리

문화누리카드 잔액은 다음 방법으로 확인할 수 있다.

- 문화누리카드 누리집
- 문화누리카드 모바일 앱
- 고객지원센터
- 카드 사용 문자
- 가맹점 결제 후 영수증
- NH농협카드 고객센터

잔액이 남아 있어도 사용 기간이 지나면 사용할 수 없을 수 있으므로 12월 31일 전까지 사용하는 것이 좋다.

## 16. 서비스에서 활용할 수 있는 판단 기준

우리 서비스에서는 문화누리카드 대상 가능성을 다음 기준으로 안내할 수 있다.

### 대상 가능성 높음

- 6세 이상
- 기초생활수급자
- 차상위계층
- 생계급여, 의료급여, 주거급여, 교육급여 수급자
- 차상위본인부담경감대상자
- 장애수당 또는 장애아동수당 수급자
- 저소득 한부모가족
- 문화생활 비용 부담이 큼
- 영화, 공연, 여행, 체육활동 욕구가 있음
- 기존 문화누리카드 발급 이력이 있음

### 확인 필요

- 수급자격 유지 여부 확인 필요
- 차상위계층 해당 여부 확인 필요
- 나이 기준 확인 필요
- 자동재충전 여부 확인 필요
- 카드 유효기간 확인 필요
- 잔액 확인 필요
- 사용 기간 확인 필요
- 가맹점 여부 확인 필요
- 추가지원금 대상 여부 확인 필요

### 대상 가능성 낮음

- 6세 미만
- 기초생활수급자 또는 차상위계층이 아님
- 수급자격이 중지됨
- 해당 연도 예산 소진으로 발급이 마감됨
- 문화·여행·체육 분야 이용 의사가 없음

## 17. 서비스 입력 항목 예시

문화누리카드 추천 또는 상담 기능을 만들 때 다음 입력 항목을 활용할 수 있다.

| 항목 | 설명 |
|---|---|
| age | 대상자 나이 |
| birthDate | 생년월일 |
| basicLivelihoodRecipient | 기초생활수급자 여부 |
| benefitType | 생계, 의료, 주거, 교육급여 구분 |
| nearPoorClass | 차상위계층 여부 |
| disabilityAllowance | 장애수당 수급 여부 |
| childDisabilityAllowance | 장애아동수당 수급 여부 |
| singleParentFamily | 저소득 한부모가족 여부 |
| selfSupportProgram | 차상위자활근로 여부 |
| medicalExpenseReduction | 본인부담경감대상자 여부 |
| previousCardIssued | 기존 문화누리카드 발급 여부 |
| autoRechargeStatus | 자동재충전 여부 |
| cardBalance | 카드 잔액 |
| cardExpirationDate | 카드 유효기간 |
| culturalActivityNeed | 문화활동 필요 여부 |
| travelNeed | 여행 필요 여부 |
| sportsActivityNeed | 체육활동 필요 여부 |
| digitalApplicationDifficulty | 온라인 신청 어려움 여부 |

이 항목들은 실제 발급 가능 여부를 확정하지 않고, 주민센터 또는 문화누리카드 누리집 신청 전 사전 확인용으로 활용한다.
